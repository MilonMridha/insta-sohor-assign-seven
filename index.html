Question :How does javascript work? (জাভাস্ক্রিপ্ট কিভাবে কাজ করে?)

Answer (উত্তর ) :
  1. জাভাস্ক্রিপ্ট হচ্ছে একটি হাই লেভেল ইন্টারপ্রেটেড প্রোগ্রামিং ল্যাঙ্গুয়েজ |
  2. একে একটি ডায়নামিক ল্যাঙ্গুয়েজ বলা হয় |
  3. জাভাস্ক্রিপ্ট তৈরী করা হয়েছিল ব্রাউজার এর সাথে communicate করার জন্য।
  4. জাভাস্ক্রিপ্ট synchronous behaviour way তে কাজ করে।
  5. এটি ব্রাউজারের v8 ইঞ্জিন দ্বারা Run করে।
  6. এইচটিএমএল এবং সিএসএস দিয়ে একটি ওয়েব পেজ ডিজাইন করে, আর সেই ডিজাইনটিকে আরো জীবন্ত করে তোলার জন্য জাভাস্ক্রিপ্ট কাজ করে।
